Github pages 设置双域名

发表于2022-01-05|更新于2022-02-09
|评论数:

原理

github pages并不支持两个域名，但我们可以让其中的一个域名绑定一个空的github pages，再通过重定向使用户转接到正确的网站

第一步

需要购买两个域名，这里我是在腾讯云服务器买的
购买域名的链接在这里

话说我这样算是在给腾讯打广告吗（rnm腾讯打钱）。

第二步

拿一个域名绑定网站的pages，然后新建一个仓库

关于域名的网站的绑定，我在这里就说一点，怎么获取你的pages的ip。

例如我的仓库名字是 old_distraged 的话,
win + r 输入cmd打开命令行，输入 

ping www.你的仓库名字.github.io -4

其中的四个长度为三的数字就是你仓库的ip啦

第三步

然后再拿一个域名来绑定这个新建的仓库

这个仓库里面有三个文件

1.CNAME

CNAME文件的创建方法就是先新建一个txt文件，然后在里面加入你的域名，然后去掉后缀的.txt

2._config.yml

文件里面只有这两行即可,要注意的是，文件的名字叫_config.yml，下划线不要少了

title: void
author: void

注意网站的域名那里，填写的是域名，不需要加www.
提交到github，找到github pages页面，选择分支，网页就生成了。

至此，就成功地完成了我们的目标，我先来演示一下，你可以访问www.金鳞星溅.top,如果你眼睛好的话，你可以短暂地看到网页的名字闪过一句话，正在重定向……
